# Dont Fluke the Flute

 ### Basic Concept 
 Dont fluke the flute is a rythm game based in the old Nokia sound and colour scheme. This was for the weekend long nokia 6 game jam in 2025 

 #### What i want to achieve in this game 
 with this game we tried to be as creative as possible with the art, theme and music to try to make an old looking but still modern rythm game for the nokia

 ### Softwares Used:

    Unity 6
      - C#
    Rider
    FL Studio
    Aesprite
